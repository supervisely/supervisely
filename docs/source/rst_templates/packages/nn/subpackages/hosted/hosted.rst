Hosted
======

**Hosted**
- All the neural network logic that is specific to interacting with Supervisely platform. Base classes for neural network training and inference with common functionality already implemented, parsing, interpreting and consistency checking of configs, class indexing (between integer IDs of the tensors and Supervisely classes) helpers.
